Notification Logger Privacy Policy

Important Notice: We highly value your privacy. All data collected by this app is stored locally on your device and will not be uploaded to any server.

1. Information Collection

The Notification Logger app collects the following types of information:

1.1 Notification Data

  • Notification Content: Including title, body text, subtext, etc.
  • App Information: Package name and app name that sent the notification
  • Timestamps: When notifications are received and removed
  • Notification Attributes: Priority, category, whether it's ongoing, etc.
  • Icon Data: Notification icons and large icons (if available)
  • Action Information: Notification action buttons and intent information

1.2 Device Information

  • App List: For blacklist management and statistics
  • Permission Status: For permission management and feature optimization

2. Information Usage

Collected information is used solely for the following purposes:

  • Recording and displaying notification history
  • Providing notification search and filtering features
  • Generating notification statistics and analysis
  • App blacklist and filtering settings
  • Improving app functionality and user experience

3. Data Storage and Security

Local Storage: All data is stored in your device's local database and will not be transmitted to external servers.
  • Data is stored locally using SQLite database
  • Data is accessible only on your device
  • All data will be completely deleted when the app is uninstalled
  • Supports manual data clearing by users

4. Required Permissions

The app requires the following permissions to function properly:

  • Notification Access: To read system notifications for recording
  • Storage: To save notification data and enable export features
  • Boot Startup: To ensure continuous operation of notification recording service
  • Foreground Service: To maintain notification monitoring service
  • Battery Optimization Whitelist: To prevent system from killing background services
  • Query Package: To get app information for blacklist management

5. Data Sharing

We do not share your data with any third parties. The sharing feature only works when you actively choose to share selected notification content through the system share function to apps of your choice.

6. User Control

You have full control over your data:

  • Enable or disable notification recording at any time
  • Set app blacklist to exclude notifications from specific apps
  • Configure automatic deletion rules
  • Manually delete any notification records
  • One-click option to clear all data
  • Uninstall the app at any time to delete all data

7. Data Retention

  • Notification data is retained according to your settings (permanent by default)
  • You can set automatic deletion rules (from 1 day to 365 days)
  • You can manually delete any data at any time
  • All data will be deleted when the app is uninstalled

8. Children's Privacy

This app is not specifically targeted at children under 13 and does not knowingly collect personal information from children. If we discover that we have collected personal information from a child, we will delete it immediately.

9. Policy Updates

We may update this privacy policy from time to time. For significant changes, we will notify you within the app. Continued use of the app indicates your acceptance of the updated policy.
